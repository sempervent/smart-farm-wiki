## Man pages

- [mosquitto](https://mosquitto.org/man/mosquitto-8.html) - running the Mosquitto broker
- [mosquitto.conf](https://mosquitto.org/man/mosquitto-conf-5.html) - the Mosquitto broker configuration file
- [mosquitto\_ctrl](https://mosquitto.org/man/mosquitto_ctrl-1.html) - command line utility for managing Mosquitto broker configuration
- [mosquitto\_ctrl\_dynsec](https://mosquitto.org/man/mosquitto_ctrl_dynsec-1.html) - `mosquitto_ctrl` batch mode for the dynamic-security plugin
- [mosquitto\_ctrl\_shell](https://mosquitto.org/man/mosquitto_ctrl_shell-1.html) - `mosquitto_ctrl` interactive shell mode (recommended)
- [mosquitto\_passwd](https://mosquitto.org/man/mosquitto_passwd-1.html) - command line utility for generating Mosquitto password files
- [mosquitto\_pub](https://mosquitto.org/man/mosquitto_pub-1.html) - command line utility for publishing messages to a broker
- [mosquitto\_rr](https://mosquitto.org/man/mosquitto_rr-1.html) - command line utility for simple request/response with a broker
- [mosquitto\_signal](https://mosquitto.org/man/mosquitto_signal-1.html) - command line utility for sending signals to a broker, most useful on Windows
- [mosquitto\_sub](https://mosquitto.org/man/mosquitto_sub-1.html) - command line utility for subscribing to topics on a broker
- [mosquitto-tls](https://mosquitto.org/man/mosquitto-tls-7.html) - brief cheat sheet for creating x509 certificates
- [mqtt](https://mosquitto.org/man/mqtt-7.html) - description of MQTT features
