# Citations

## Raw citations

Link directly to stable paths under `raw/`:

```markdown
See [processed note](../../raw/processed/2026/example-llm-wiki-note.md).
```

## `source_ids`

Use `source_ids` in frontmatter for machine-readable linkage (paths or opaque ids consistent in your vault).

## Wiki vs raw

Do not present wiki prose as verbatim source text without quoting. Prefer short quotes with pointers for high-stakes claims.
