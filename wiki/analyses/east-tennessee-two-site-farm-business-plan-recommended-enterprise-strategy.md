# Recommended enterprise strategy and rationale

**Hub**: [`Planning framework`](east-tennessee-two-site-farm-business-plan-framework.md)  
**Evidence**: [`Enterprise options analysis — 120-acre East Tennessee two-site smart farm`](east-tennessee-two-site-farm-business-plan-enterprise-options-analysis.md)

## Strategy family (chosen)

**Phased mixed enterprise** with **grazing-led commercial beef** as the **primary** 120-acre engine, **optional lease / contract income** as **stabilizer**, and **deliberate deferral** of high-labor or long-payback enterprises (specialty veg, orchard-primary, agritourism-primary) until **Phase 3+** **if** gates clear.

| Layer | Content |
|-------|---------|
| **Primary** | **Cow-calf** *or* **stocker/backgrounder** on improved pasture (pick after Phase 1 carrying-capacity + market probe) |
| **Stabilizer** | **Cash rent**, **hunting lease**, **custom grazing**, or **hay/custom hire** (not all)—chosen for **low trip burden** |
| **Deferred** | Specialty crops, bearing orchards, event-led agritourism—**optional** slices, not foundation |
| **Homestead (5 ac)** | **Command / telemetry / workshop**—**not** primary commercial production |

## Rationale (why this family)

| Criterion | Fit |
|-----------|-----|
| **Profit first** | Grazing uses **land** as factory; margin is **thin** but **scalable** with discipline and **premium/direct** optional layer |
| **Low labor** | **Batchable** herd work vs **daily** row-crop hand labor; **35 min** penalizes daily-site crops |
| **Resilience** | **Grass + livestock** hedges **crop** weather/market for a **family** without duplicate equipment lines |
| **Automation ROI** | **Water, pump, gate, power** telemetry maps to **real trip reduction**; avoids “automation theater” |
| **Capital phasing** | **Fence/water** are **sequenced**; **lease** can **fund** improvements without **heroic** livestock exposure on day one |
| **Optionality** | **Land-first** sub-phase inside Phase 1 preserves **exit** if gates fail |

## What this strategy is not

- **Not** a commitment to **cow-calf** before **Phase 1** shows **forage + water + market** support **stocker** vs **cow-calf**.
- **Not** ignoring **orchard/specialty/agritourism**—they are **explicitly** **late** unless **validation** proves **fast** margin (unusual).

## Branching logic (after Phase 1)

| If Phase 1 shows… | Prefer… |
|-------------------|---------|
| Strong **grass**, weak **facilities** | **Stocker** or **custom graze** until **corral/water** done |
| Strong **facilities**, family wants **breeding** | **Cow-calf** with **disciplined** culling |
| **Thin** cattle margins but **solid** lease rates | **Higher** lease share, **smaller** owned herd |
| **Water** unreliable | **Do not** scale herd—**fix water** (Phase 2 gate) |

## Phasing tie-in

| Phase | Enterprise content |
|-------|-------------------|
| **0** | No livestock commitment; clarify **use**, **boundaries**, **stop rules** |
| **1** | **Land intelligence** + optional **lease**; **pilot** telemetry on **one** system |
| **2** | **First** owned grazing **unit** (herd or contract stack) |
| **3** | **Scale** herd, **add** secondary stream **only** if labor + margin tests pass |
| **4** | **Salary-replacement** **mix**: margin **+** possible **direct** sales **+** stabilized asset use |

## Known / assumed / open

| Class | Items |
|-------|--------|
| **Known** | Two-site structure; **profit > labor > resilience** stack; **120 ac** enterprise undecided **before** this package—now **narrowed** to **family** above |
| **Assumed** | **No** full-time resident manager at **120** unless later **budgeted**; **family** can batch **3–4×/week** **minimum** checks in **Phase 2+** |
| **Open** | **Stocking rate**; **auction vs direct**; **lease** terms; **processing** access; **organic/grass-fed** premium **real** in **your** zip |
