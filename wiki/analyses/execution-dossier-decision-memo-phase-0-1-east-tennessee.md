# Decision memo — Phase 0–1 execution (East Tennessee two-site)

## Purpose

One memo for: **what the family may do now**, **what is pilot-only**, and **what is forbidden until validation gates pass**. Execution means **pilot-grounded** action—not premature scale.

**Gates**: [`Validation backlog`](east-tennessee-two-site-farm-business-plan-validation-backlog.md) (G1–G3), [`Validation — before major spend`](validation-backlog-before-major-spend-two-site-smart-farm.md) (G1–G8 where applicable).

**Governance / doctrine ledger**: [`Business plan integration revision audit (2026-04)`](east-tennessee-two-site-farm-business-plan-integration-revision-audit-2026-04.md) (what the **canonical plan** now assumes about evidence vs execution). **Privileged remote access** stays behind [`Observability, secrets, and trust bar — homelab / farm edge`](observability-secrets-and-trust-bar-homelab-farm-edge.md) **trust bar**—not “VPN exists.”

**Execution gates (EG)**: [`Execution gates — Phase 0–1 (East Tennessee two-site)`](execution-gates-phase-0-1-east-tennessee-two-site.md) — **pilot allowed** vs **production allowed** by domain; use **before** widening pilots or claiming **G1** passed.

---

## Approved for action now (no gate waiver)

**Definition**: Low cash outlay; reversible; does **not** commit herd scale, perimeter fence MVP, financed equipment, or remote actuation.

| Category | Allowed |
|----------|---------|
| Governance | Stop rules posted; hour log started; one canonical map authority file |
| Land intel | NRCS pulls; walks and photos; segment notes |
| Legal / use | County / zoning calls; scoped attorney consult (fee per budget) |
| Markets | Sale bill collection; processor calls; haul math |
| Water | Manual check logs; water points on map |
| Access | Bad-weather drives; photo logs |
| Telemetry | Manual baseline logs (G8 path); cell screenshots at pins; read-only comms tests |
| Labor | Weekly rhythm doc; two-site trip purpose logging |

---

## Approved only for pilot (bounded scope + kill criteria)

**Definition**: Small dollar or time box; owner named; kill criteria written before start; no implicit scale.

| Category | Pilot rules |
|----------|-------------|
| Telemetry | One sensor class + one uplink path—[`Instrumentation priority matrix`](instrumentation-priority-matrix-two-site-smart-farm.md); stop if triage exceeds benefit per [`Automation stop rules`](automation-stop-rules-two-site-smart-farm.md) |
| Business | Market probes only until V5/V6 have first-pass evidence; small/short lease pilot if aligned to [`Recommended enterprise strategy`](east-tennessee-two-site-farm-business-plan-recommended-enterprise-strategy.md) |
| Infra | Spot fence repairs for safety/escape only—not full perimeter MVP without G1 |
| Labor | Surge window test with [`coverage matrix`](family-labor-model-and-coverage-matrix-two-site-smart-farm.md) update |

---

## Forbidden until validation gates pass

| Forbidden | Until |
|-----------|--------|
| Full perimeter fence MVP / major water development CAPEX | G1 + [`major-spend`](validation-backlog-before-major-spend-two-site-smart-farm.md) rows (water rights G3 as applicable) |
| Owned herd above documented conservative capacity | V4 + G1 + V6 path credible |
| Financed equipment (loader, hay line, truck class) | Hours/year proof + custom-hire quote per [`Remediation` §3](east-tennessee-two-site-farm-business-plan-remediation-backlog.md); V9 as applicable |
| Telemetry fleet, multi-zone cameras, remote actuation | G8 manual baseline; PA-* proofs for actuation ([`Smart tech strategy`](east-tennessee-two-site-farm-business-plan-smart-tech-strategy.md)); NS-* stop rules |
| Second major enterprise stream | V11 surge covered + margin proof (Phase 3-class—see [`validation backlog`](east-tennessee-two-site-farm-business-plan-validation-backlog.md) G2) |
| Phase 4–style salary replacement claims | G3 per validation backlog |

**Insurance illusion**: A **quote** or **binder PDF** is **not** proof of coverage. **V9** **pass** means **reviewed** with the agent for **named** **perils**, **exclusions**, **equipment** **limits**, and **umbrella** **layer**—not a checkbox.
