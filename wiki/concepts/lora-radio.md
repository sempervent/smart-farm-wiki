# LoRa (radio)

**LoRa** usually refers to the **proprietary chirp spread spectrum physical layer** used for long-range, low-rate RF links. **LoRaWAN** is the common upper-layer star network protocol built on LoRa; **Meshtastic** uses LoRa radios with a different mesh stack (see [`Meshtastic`](meshtastic.md)).
