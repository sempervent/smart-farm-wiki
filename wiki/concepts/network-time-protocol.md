# Network Time Protocol (NTP)

**NTP** is a widely deployed internet protocol for synchronizing system clocks over packet networks, typically to millisecond-scale accuracy for general-purpose hosts—contrasted with [`Precision Time Protocol (PTP)`](precision-time-protocol.md) when sub-millisecond or nanosecond discipline is required.
