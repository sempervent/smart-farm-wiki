# Source note: NIST Internet Time Service — server list (snapshot)

**Raw path:** [`raw/processed/2026/nist-internet-time-service-servers.md`](../../raw/processed/2026/nist-internet-time-service-servers.md)

**Summary**

Snapshot of the **NIST Internet Time Service (ITS)** documentation: `time.nist.gov` round-robin usage, per-hostname **IPv4/IPv6** endpoints (Gaithersburg, WWV Boulder-area, etc.), encouragement to prefer **NTP** over legacy TIME/DAYTIME, **query rate** guidance (not more than once per ~4 seconds), authenticated NTP endpoints, and **UT1** NTP server note. IP assignments can change—verify on the live NIST page before hard-coding devices.
